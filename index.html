Some packages to help with OpenGL content on Google's go

This organization hosts gl and glfw3, two very useful packages for making cross-platform graphics applications. There are some helper libraries such as glu and glh, if you want help getting started drawing some basic primitives, using framebuffers or shaders.

A good starting point would be to take a look at the examples for the glfw3 project.

Need help or want to contribute?

We can be found on #go-gl on freenode, where you can ask any questions (though you might have to idle a while before getting a response) or help in answering them. Graphics help can also be found on #opengl.
